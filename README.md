# My Task Manager

## Description

This project is a responsive task completion website built using Next.js (App Router) and Tailwind CSS.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- TypeScript

## Installation
Note: You must first create your own generic Next.js application, then copy and paste the contents of this folder into the directory.
1. cd my-task-manager
2. npm run dev
